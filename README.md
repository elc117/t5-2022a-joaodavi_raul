## Objetivo

O objetivo do trabalho é a criação de um projeto inicial de um jogo que contenha o básico para uma pequena gameplay, utilizando a engine Unity de forma a aplicar conceitos e técnicas aprendidos relacionados à programação orientada a objetos. Anexaremos a este read.me alguns vídeos e explicações do que foi realizado, além de um link para o repositório onde os arquivos principais do jogo estão.
